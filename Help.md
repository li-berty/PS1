# Cheat sheet
**Important commands**  
To get help on a cmdlet *__get-help__*:  
`Get-Help Get-Service`  
List of all available cmdlets *__get-command__*:  
`Get-Command`  
All properties and methods of an item *__get-member__*:  
`Get-Service | Get-Member`
